# Nakshatra

Cursor project for the **Infinity Nakshatra Society** resident and maintenance portal.

| Item | Path |
|------|------|
| Root | `C:\HC\Reports\cursor\Nakshatra` |
| Portal | `infinity_nakshatra_dashboard.html` |
| Apps Script | `apps_script_collection_data.gs` |
| Setup | `COLLECTION_SETUP.md`, `How to Use Project.txt` |
| **Deploy from Cursor** | **`DEPLOY_FROM_CURSOR.md`** |
| GitHub | https://github.com/infinitynakshatra/nakshatra-portal |

Open in Cursor via **`Nakshatra.code-workspace`** so the workspace label shows as **Nakshatra**.

**Sync commands:** `npm install` once, then `npm run apps-script:deploy` (Google) or `git push origin main` (GitHub). See **`DEPLOY_FROM_CURSOR.md`**.
